# tms-min-side
Container som setter sammen mikrofrontends for Min Side.

## Kom i gang
1. Bygg tms-min-side ved å kjøre npm run build
2. Start appen lokalt ved å kjøre npm run dev
3. Appen nås på http://localhost:3000/minside
